## 📍 Onde ocorre

[https://platform-stg-userin-ai.fly.dev/journey-builder/new](<https://platform-stg-userin-ai.fly.dev/journey-builder/new>)

## 🔁 Passo a Passo

1. Entrar na plataforma
2. Ir em Jornadas -> Builder -> Criar uma nova jornada
3. Criar um fluxo com um gatilho de regra para exibir um modal
4. Criar um outro fluxo com um gatilho manual para exibir um mini game

## ❌ Resultado Atual

A jornada segue sempre o mesmo fluxo.

## ✅ Resultado Esperado

A jornada deve seguir o fluxo de acordo com o gatilho que foi acionado. Se os dois gatilhos forem acionados, deve se seguir os dois fluxos.
