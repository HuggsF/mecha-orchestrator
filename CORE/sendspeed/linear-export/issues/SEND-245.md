## Descrição

**Como Analista de Produto**
**Quero visualizar e editar o título, descrição e forma de conversão do card**
**Para garantir que cada card esteja configurado corretamente e, caso não exista uma conversão personalizada, criar uma na hora**

**Critérios de Aceite:**

* Deve existir uma página dedicada com os campos: título, descrição e forma de conversão.
* Se o card **não tiver conversão associada**, deve aparecer a opção **"Criar Conversão Personalizada"**.
* Ao criar a conversão, o sistema deve salvar e já vincular automaticamente ao card.
* A forma de conversão deve permitir seleção de uma lista existente ou criação na hora.
* O usuário deve conseguir salvar alterações no título e na descrição.
* O sistema deve validar campos obrigatórios antes de salvar.
* Após salvar, o card deve exibir a conversão selecionada/criada imediatamente.

---

## **2) Vincular o card a uma ou mais jornadas**

**Como Analista de Produto**
**Quero vincular um card a uma ou mais jornadas**
**Para que o mesmo card possa ser reutilizado e analisado em diferentes fluxos da plataforma**

**Critérios de Aceite:**

* Dentro da jornada, deve existir a opção de **associar um card existente**.
* O card deve permitir **vínculo com múltiplas jornadas** ao mesmo tempo.
* O sistema deve exibir uma lista de jornadas disponíveis para seleção múltipla.
* Ao selecionar as jornadas, o vínculo deve ser salvo e refletido imediatamente.
* O card deve mostrar em quais jornadas está vinculado.
* O sistema deve impedir vínculos duplicados.
* Desvincular o card de uma jornada não deve afetar os demais vínculos.

---

## **3) Selecionar a jornada para análise dentro do Analytics**

**Como Analista de Produto**
**Quero selecionar uma jornada específica dentro do Analytics da plataforma**
**Para analisar métricas, conversões e performance segmentadas por jornada**

**Critérios de Aceite:**

* O módulo Analytics deve exibir um seletor com todas as jornadas disponíveis.
* O usuário deve poder selecionar **uma jornada por vez** para análise.
* Ao selecionar uma jornada, todos os gráficos e métricas devem atualizar automaticamente.
* Deve existir um estado default quando nenhuma jornada for escolhida (ex.: visão geral).
* O sistema deve indicar claramente qual jornada está sendo analisada no momento.
* Os filtros adicionais do Analytics devem continuar funcionando dentro da jornada selecionada.
* Caso a jornada selecionada seja excluída, deve voltar para o estado default.
