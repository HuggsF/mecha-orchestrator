## Descrição

> **Como** analista de produto
>
> **Quero** poder selecionar algumas regras e deixa-las ativadas obrigatóriamente
>
> **Para** facilitar o rastreio de algumas informações através das regras.

---

# Critérios de aceite:

* A regra precisa ter um botão para selecionar se esta regra aparece ou não para o cliente.
* A regra precisa ter um botão de ativação que será o botão que ativará a regra obrigatória na conta do cliente.
* Quando ativado, precisa aparecer para quais empresas queremos adicionar.
* A regra não pode ser editada pelo cliente, apenas a visualização quando escolhido.
* Exemplo de regras: Modar_Register_Show, Modal_Register_Close, Var login.

# Cenários de teste:

* Entrar na plataforma.
* Criar uma regra obrigatória.
* Ativar ela.
* Selecionar a empresa.
* Selecionar para aparecer pro cliente.
