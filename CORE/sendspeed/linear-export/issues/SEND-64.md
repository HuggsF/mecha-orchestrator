## Descrição

**Como** administrador da plataforma, 
**Quero** escolher quais opções do menu lateral cada cliente vê (ex.: só "Início" e "Journey", ou "Início, Journey, Companion e Configurações") 
**Para** entregar a experiência certa para cada cliente sem causar problemas nos demais.

**Critérios de aceite (testáveis)**

* Para um cliente específico, **consigo marcar/desmarcar** as opções que devem aparecer no menu.
* Ao entrar como esse cliente, **apenas as opções escolhidas** ficam visíveis.
* **Alterar um cliente não muda** o menu dos outros.
* Existe um **menu padrão** para novos clientes, que posso trocar depois.
* **Posso salvar e editar** a configuração com segurança e ter um **histórico simples** de quem alterou e quando.
* Se algo der errado, **consigo voltar** para a configuração anterior rapidamente.
