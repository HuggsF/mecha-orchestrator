## Descrição

Substituir o componente de calendário atual por um novo componente seguindo o padrão definido no Figma. Após implementação, replicar para todas as telas que usam seleção de data.

[Figma Reference](https://www.figma.com/design/dXUMDRphe6NtNiDo0pYp5l/UserIn-App?node-id=386-3898)

## Funcionalidades

* **Períodos pré-definidos:** Últimos 7/30/90 dias, semana/mês/ano passado, trimestre, ano
* **Duplo calendário:** mês atual + seguinte lado a lado
* **Navegação:** setas por mês + clique no título para salto rápido
* **Seleção visual:** clique início + fim, intervalo destacado em azul
* **Botões:** Aplicar / Cancelar
