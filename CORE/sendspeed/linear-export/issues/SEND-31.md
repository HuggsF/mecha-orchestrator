## Descrição

**Como** Especialista em CRO (Conversion Rate Optimization)

**Eu quero** ver a análise detalhada de um card específico com preview e métricas organizados

**Para que** eu possa identificar gargalos no funil de conversão e otimizar a performance dos cards de forma data-driven

**Critérios de Aceitação:**

- [X] Devo ver o preview visual do card em um bloco dedicado
- [X] Devo ver as principais métricas em cards organizados (2x2): Impressões, ~~Preview~~ Clicks, CTA, ~~Conversões~~
- [ ] Devo ver o funil completo centralizado com análise de evasão. \~ Para ver funil completo é necessária métrica de Conversão
- [X] Cada etapa do funil deve mostrar quantidade, taxa de conversão e perdas
- [ ] Taxa de conversão geral deve ser exibida de forma minimalista \~ Para visualizar taxa de conversão é necessária definição de métrica
- [ ] Tempos médios entre etapas devem ser exibidos 
- [X] Breakdown por dispositivo deve estar disponível
- [ ] Análise de usuários anônimos vs identificados deve ser mostrada

### Tracker

Responsável por atribuir os acessos/visualizações etc no card exibido
Só que terá uma função exclusiva pra isso. Gravando na mesma tabela.
