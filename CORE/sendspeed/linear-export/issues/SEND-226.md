## Descrição

### Como head de produto

### Quero ter uma rota dedicada com Alpha Sender configurado exclusivamente para os disparos dos clientes Donald e Bet.bet

### Para garantir maior entregabilidade, reconhecimento de marca e prioridade no tráfego de mensagens

---

### **Critérios de Aceite:**

* Rota dedicada criada no provedor de SMS, vinculada ao cliente premium.
* Configuração do Alpha Sender aprovada e validada (ex: "LIDERBET" / "BET.BET").
* Teste de envio realizado com sucesso (mínimo 3 mensagens em diferentes).
* Logs de entrega e status report configurados corretamente.
* Cliente notificado sobre o funcionamento e prazos de ativação.
* Documentação interna atualizada (rota, ID do cliente, endpoint e credenciais).

---

### **Notas Técnicas / Contexto (opcional):**

* O Alpha Sender será provisionado junto à operadora principal.
* A rota dedicada deve ser vinculada ao sistema atual via API REST de disparo.
* Caso o cliente tenha múltiplas origens, manter fallback na rota padrão.
* SLA de ativação: até 48h úteis após confirmação de dados com a operadora.
