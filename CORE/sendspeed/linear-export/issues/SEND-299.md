## Descrição

> Como PO
>
> Quero integrar o sistema de Modal reformulado
>
> Para ter um sistema de configuração mais ampla diretamente na plataforma.

---

# Use Case:

* Vini ira acessar a plataforma, criar um modal e fazer ele aparecer no site do cliente através do sistema de regras.

# Critérios de aceite:

* O sistema de regras devem estar funcionais.
* A tela antiga de modal deve ser removida ou substituída pela nova.
* Todos os componentes devem estar funcionais tanto para Desktop quanto para Mobile.
* A trava por empresa deve estar funcional, deve estar protegido caso puxe por outra empresa.
* Toda vez que eu der um GET ele deve estar cacheado, caso altere alguma coisa ele deve validar através do cache.
* Os analytics que acompanham devem estar funcionais.
* A tela deve ser integrada e estar totalmente funcional juntamente com todas as configurações integradas dela.
